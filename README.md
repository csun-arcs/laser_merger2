# laser_merger2
Muilti laser merger to pointcloud2 and laserScan for ROS2.

## Built with

- ROS Foxy under Ubuntu 20.04 LTS

------

## Getting Started

### Installation

``` bash
$ git clone https://github.com/qaz9517532846/laser_merger2.git
```

### Run

------

``` bash
$ ros2 run laser_merger2 laser_merger2.launch.py
```
